bookshelf planner

数年前に東京に引っ越して以来 11.0m2 の 1R に住んでいる。部屋が狭いため収納が問題である。隙間なく家具を並べるには、ちょうど良いサイズのものを探さなければならない。

そんな中、友達から棚の作り方を教わった。流れとしては、まず欲しい棚を設計する。そこから必要な板のサイズを調べる。ホームセンターに行き、大きな合板を購入し、板をカットしてもらう。最後にネジやくぎを駆使して組み立てる。さっそく近所の東急ハンズでラワン合板を調達し、欲しいサイズの棚を作った。木ネジを使って固定したので、引っ越しする時も分解して組み立て直すことができる。

ところで、作りたい棚に対してどのサイズの合板を何枚購入するか少し悩んだ。たとえば東急ハンズでは

  • 300 x 450 [mm]
  • 450 x 600 [mm]
  • 450 x 910 [mm]
  • 910 x 910 [mm]
  • 910 x 1,820 [mm]
  • ...

と様々なサイズの合板があるため、どのように選べばコストを抑えられるか少し考えなければならない。また合板自体の厚さやカット時に約 3mm 板が削られることも考慮する必要がある。このような問題をブラウザ上で計算する仕組みをつくれば、もしかしたら楽しいかもしれない。欲しい棚を three.js で描画し、収納したい物のサイズや棚自体のコスト等の制約条件をユーザに選んでもらって、自動的に計算してくれるもの。たぶんナップサック問題などの線形（整数？）計画問題の solver が使えるのではないかと思う。試しに調べてみると JavaScript で書かれた Simplex method の実装 を見つけた。棚の数が異常に多いエグい本棚なども解くことができれば楽しそうだ。そのうち作ってみようと思う。

MOA Museum of Art

熱海のMOA美術館に先日行ってきた。熱海駅から美術館入口までバスで約10分。会場にはそこから7機のエスカレータを登ってようやくたどり着くことができる。美術館からは熱海の景色を一望することができる。常設展には国宝を含む日本や古代中国などの作品が展示されている。

平松礼二・森口邦彦展が開催されていた。森口邦彦の作品がとくに面白かった。森口邦彦は友禅染めの重要無形文化財保持者（人間国宝）であり、街で時々見かける三越のショッピングバッグも手がけたそうだ。それまで極彩色の花が散りばめられたような着物しか知らなかったので、オプ・アートを思わせる幾何学模様をふんだんに取り入れた着物はとても新鮮だった。

また会場では制作風景を撮影したビデオも上映されており、その中で紹介されていた蒔糊（まきのり）という技法が印象に残った。これは布の上に糊を乾燥させた小さな欠片をまいて、その上から染めることにより欠片以外の部分が染め上がるというものである。欠片のまき方を工夫することで、グラデーションなど様々な表現が可能である。

Moi!

GitHub上にブログを作成した。趣味で作成しているプログラムの作業進捗やレビューを載せようと思う。

ページ構築には Python で記述された Pelican なる static site generator を使った。 Theme は Blue Penguin に手を少し加えて適用した。

タイトルの Pikku-Pukki とはフィンランドのトゥルクで見つけた島の名前である。そのまま「ピック・プッキ」と発音する。 pikku は「小さな」 pukki は「雄のヤギ」を意味する。リズミカルだしアナグラムでもあるので、名前を拝借した。
