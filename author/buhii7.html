bookshelf planner 4

ortoolpy を参考に１枚板のギロチンカットを解くプログラムを書いてみた（packer.py）。PlotDevice で以下の様な画像を描画する。

問題についてもう少し調べた。今回の問題は two-dimensional bin packing problem (2DBPP)、その中でも複数の種類のコンテナを扱う 2D variable-sized bin packing problem (2DVSBPP) と呼ばれているもののようだ。"A hybrid heuristic algorithm for the 2D variable-sized bin packing problem" という論文を見つけた。この論文をもとに実装していけば良さそうだ。

2015-10-03

梨

先週から引き続き咳が続いている。薬は一応効いているが、もうしばらく経っても治らないようだったら、病院に行こう。多少マシになっている気はするけど。家で梨を剥いて食べた。

本屋に寄ったら結晶群という本があったので購入。美の幾何学という本を以前読んだとき「平面を埋め尽くす連続模様のパターンは17通りある」ということを知ったが、この本はオービフォールドなるものを駆使して証明したりしている。平面の模様（平面結晶群）の話をしているのに、トーラスとか丸い立体が出てくるので不思議。アルハンブラ宮殿のアラベスクには17種類すべて含まれているそうだ。数学は得意では無いので読むのに時間がかかりそうだが、図がたくさん含まれているのでなんとか読めるかなぁ...
