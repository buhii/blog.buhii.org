pikku pukki

2015-09-22

Moi!

GitHub上にブログを作成した。趣味で作成しているプログラムの作業進捗やレビューを載せようと思う。

ページ構築には Python で記述された Pelican なる static site generator を使った。 Theme は Blue Penguin に手を少し加えて適用した。

タイトルの Pikku-Pukki とはフィンランドのトゥルクで見つけた島の名前である。そのまま「ピック・プッキ」と発音する。 pikku は「小さな」 pukki は「雄のヤギ」を意味する。リズミカルだしアナグラムでもあるので、名前を拝借した。
